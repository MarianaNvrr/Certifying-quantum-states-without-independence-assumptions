This repository contains the code used to reproduce the results presented in *Certifying quantum states without independence assumptions* ([arXiv:2606.31913](https://arxiv.org/abs/2606.31913)).

- `main.jl`: Contains the auxiliary functions used by the numerical simulations, including state preparation, Pauli observables, drift-source models, and Hamiltonian construction.

- `hamiltonian.jl`: Runs the numerical simulations for Hamiltonian certification. The code simulates non-i.i.d. sequences of quantum states generated by a drift model and certifies the average energy of the use states from the randomly selected test states.

- `witness.jl`: Runs the numerical simulations for entanglement-witness certification. It includes the dynamic certification and a static benchmark protocol.

- `experiment_fidelity.ipynb`: Contains the code used for the fidelity-estimation experiment with an IBM quantum processor. The notebook prepares the circuits, submits the jobs, retrieves the data, and postprocesses the results.

- `data_extraction.jl`: Reproduces the figures shown in the paper from the data stored in the `data/` folder.
